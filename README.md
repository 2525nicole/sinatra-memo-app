課題名: Sinatra を使ってWebアプリケーションの基本を理解する

# sinatra_memo_app

## 説明
このアプリを使えばいつでもメモを登録、確認、削除することができます。

登録済みのメモの内容編集も可能です✏️

## 利用手順

1. このリポジトリをクローンする

`$ git clone git@github.com:2525nicole/sinatra-memo-app.git`

2. リポジトリに入る

`$ cd sinatra-memo-app`

3. Gemをインストールする

`$ bundle install`
